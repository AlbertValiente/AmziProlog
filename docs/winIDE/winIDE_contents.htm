Original Amzi! Windows IDE

The original Amzi! Windows IDE (WinIDE) is a relatively simple IDE ideal for introductory and student use in learning Prolog. It is included in the Amzi! distribution as an alternative to the full-featured Eclipse IDE which, while being much more powerful, takes a bit longer to learn.
